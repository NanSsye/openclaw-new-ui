### 2. 附件链路

  - 外部常见文件链接可显示为附件
- 用户上传附件、AI 回传附件展示链路已打通

### 3. 登录 / 设置 / 网关连接

- 登录页改为真实验证网关连接
- 账户设置页应用配置前先验证连接
- `password` 已接入完整链路
- 修复 Gateway `hello` 双触发
- 修复 Gateway 报错只显示 `{}` 的问题

### 4. React / 工程质量

- 修复多处 `set-state-in-effect`
- 修复多处 render 期间读取 ref
- 清理 `.dockerignore` / `.gitignore` / eslint ignore
- 清理多个页面的 `any`、依赖、未使用代码问题
